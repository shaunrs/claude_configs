# Library Usage

## Libraries Over Custom Code

- **Prefer established libraries** - Use well-tested libraries over custom implementations for solved problems. Libraries handle edge cases you haven't thought of.
- **Don't reinvent the wheel** - If a mature library exists for your use case, use it rather than writing custom code.

## Version Management

- **Upgrade over workaround** - When up-to-date documentation references methods or parameters that don't exist in your application, prefer upgrading the library over working around the problem.
